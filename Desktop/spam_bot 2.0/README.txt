при запуске бота создается:
- база данных с пользователем, который запустил бота
- БД пустая, куда в будущем добавляются ники юзеров и так же удаляются
- БД, где храниться текст сообщения

Есть возможность добавлять в БД контакты и удалять их оттуда
При нажатии кнопки "рассылка" осущетсвляется отправка сформированного сообщения пользователям
